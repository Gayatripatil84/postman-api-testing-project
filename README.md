# Postman API Testing Project

This project demonstrates API testing using Postman.

## Features
- GET request
- POST request
- PATCH request
- DELETE request
- Query parameters
- JSON request body
- API Key authentication
- Postman test scripts
- Collection variables

## APIs Tested
- Get all books
- Get fiction books
- Get book by ID
- Add new book
- Checkout a book
- Delete a book

## Tools Used
- Postman
- REST API
- JavaScript (for test scripts)

## Project File
The Postman collection file is included in this repository and can be imported directly into Postman.
